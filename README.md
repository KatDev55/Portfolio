# 🌟 Modern Personal Portfolio

![Banner](https://via.placeholder.com/1200x400.png?text=Modern+Personal+Portfolio)

ยินดีต้อนรับสู่โปรเจกต์ **Personal Portfolio** เว็บตำแหน่งผลงานส่วนตัวที่ออกแบบมาเพื่อให้แสดงผลได้อย่างพรีเมียม, ทันสมัย, และทำงานได้อย่างลื่นไหลในทุกมุมมอง ไม่ว่าจะเป็นคอมพิวเตอร์ หรือ มือถือ 

เป้าหมายหลักของโปรเจกต์นี้คือการผสมผสานระหว่าง **ดีไซน์ที่สวยงาม (UI)** และ **ประสบการณ์การใช้งานที่ไร้รอยต่อ (UX)** ผ่านโครงสร้างโค้ดที่สะอาดและง่ายต่อการทำงานเป็นทีม

---

## 🚀 ฟีเจอร์หลัก (Key Features)

- 🌗 **Dark / Light Mode**: ปรับเปลี่ยนธีมหน้าเว็บให้เข้ากับสายตาผู้ใช้งานได้อัตโนมัติ 
- 🎯 **Smooth Scroll & Scroll Spy**: ระบบเลื่อนหน้าจอแบบนุ่มนวล พร้อมแถบ Navbar ที่จับตำแหน่งเนื้อหา (Active Menu) ให้ตรงกับหน้าจอที่ผู้ใช้งานกำลังอ่านแบบ Real-time
- 📱 **Fully Responsive**: จัดรูปแบบตารางและเนื้อหาอัตโนมัติเพื่อให้ดูดีที่สุดไม่ว่าจะเปิดในมือถือหรือหน้าจอใหญ่
- ✨ **Micro-Animations**: แอนิเมชันแบบจุดต่อจุด (ปุ่มกด, การเลื่อนเมาส์, การโหลดหน้าจอ) ที่ให้ความรู้สึกมีชีวิตชีวา

## 🛠️ เหตุผลที่เลือกใช้เทคโนโลยีเหล่านี้ (Tech Stack & Rationale)

โปรเจกต์นี้ถูกสร้างขึ้นมาอย่างตั้งใจด้วยชุดเทคโนโลยี (Tech Stack) ที่เหมาะสมที่สุดสำหรับงานยุคใหม่ ทั้งในมุมของนักพัฒนา (DX) และผู้ใช้งานจริง (UX):

1. **[React.js]** - *Core Library* 
   - **ทำไมถึงใช้?**: การแยกส่วนหน้าเว็บเป็นชิ้นๆ (Component-based) ทำให้หน้าเว็บทำงานแบบไร้รอยต่อโดยไม่ต้องรีโหลดหน้าใหม่ (SPA) ทำให้โค้ดแก้ไขง่ายและเป็นระเบียบเมื่อโปรเจกต์ใหญ่ขึ้น
2. **[Tailwind CSS]** - *Styling*
   - **ทำไมถึงใช้?**: ลดเวลาในการเขียน CSS ทั่วไปลงมหาศาล และทำให้การทำเว็บที่รองรับมือถือ (Responsive) เป็นเรื่องง่ายเพียงแค่พิมพ์คลาสคำสั่งลงไป รวมถึงการจัดการระบบ Dark Mode ก็ทำได้ในไม่กี่ตัวอักษร
3. **[Framer Motion]** - *Micro-Animations*
   - **ทำไมถึงใช้?**: ให้สัมผัสความพรีเมียม (Premium Feel) ของปุ่มกด นำเสนอแอนิเมชันที่มีฟิสิกส์เหมือนจริง (คลิกแล้วยุบ, โฮเวอร์แล้วขยาย) ซึ่งหาไม่ได้จาก CSS ธรรมดา
4. **[AOS (Animate On Scroll)]** - *Scroll Animations*
   - **ทำไมถึงใช้?**: ทำให้เนื้อหาดูมีมิติโผล่ขึ้นมาตามจังหวะที่ผู้ใช้งานเลื่อนหน้าจอลงมา เพิ่มระดับความน่าสนใจให้กับข้อมูลแทนที่จะแสดงผลทื่อๆ
5. **[Lucide React & React Icons]** - *Iconography*
   - **ทำไมถึงใช้?**: ไอคอนตระกูลคมชัดสูง (SVG) ไม่แตกเมื่อซูม และปรับสีตามธีมและข้อความได้อัตโนมัติ

## 📁 โครงสร้างโปรเจกต์เพื่อการทำงานเป็นทีม (Folder Structure)

เพื่อให้การเอาโค้ดขึ้น GitHub และการพรีเซนต์ให้เพื่อนร่วมทีม (หรือแม้แต่ Client) เข้าใจง่ายที่สุด เราขอแนะนำโครงสร้างต่อไปนี้:

```text
src/
 ├── assets/        # รูปภาพ, ไฟล์โลโก้, กราฟิกหลัก
 ├── components/    # (UI Components ที่ใช้ซ้ำได้)
 │   ├── ui/        # ปุ่ม, การ์ดโปรเจกต์, Badge
 │   ├── layout/    # Navbar.jsx, Footer.jsx, MainLayout.jsx
 │   └── sections/  # ชิ้นส่วนหน้าจอ: Hero.jsx, About.jsx, Projects.jsx
 ├── constants/     # 💡 โฟลเดอร์เก็บข้อมูล (Data) เช่น ประวัติส่วนตัว, รายการเมนู
 │   ├── siteData.js# ตัวหนังสือต่างๆ ที่ปรากฏบนเว็บ
 │   └── works.js   # ข้อมูลโปรเจกต์ (ภาพ, ลิงก์, แท็ก) 
 ├── hooks/         # ฟังก์ชันลับเบื้องหลัง เช่น useScrollSpy.js 
 ├── App.jsx        # จุดเชื่อมตัวโครงสร้างเว็บ (Root Element)
 └── index.css      # ค่าคอนฟิก Tailwind และธีมพื้นฐาน
```

## 📝 สำหรับ Client / เพื่อนร่วมทีม (How to Customize without coding)

โปรเจกต์นี้ถูกออกแบบมาให้ **"การแก้ไขข้อมูลส่วนตัว"** ทำได้ง่ายโดยไม่ต้องแตะต้องกระบวนการเรนเดอร์ React เพียงแค่เปิดไฟล์ฐานข้อมูลในโฟลเดอร์ `constants/` (หรือ `Data/`) เท่านั้น:

1. **เปลี่ยนรายการโปรเจกต์**: เปิดไฟล์ `src/Data/ProjectsData.js` แล้วแก้ไขรูปภาพ ลิงก์ และคำอธิบาย
2. **เปลี่ยนข้อมูลติดต่อ/ลิงก์**: ค้นหาตัวแปรลิงก์เพื่อเปลี่ยนแปลงช่องทางการติดต่อได้เลยทันที ระบบจะอัปเดตเองโดยอัตโนมัติ

สิ่งนี้ช่วยแยกระหว่าง **ดีไซเนอร์/ฝั่งจัดการเนื้อหา** กับ **ฝั่งเขียนโค้ด (Dev)** ออกจากกันอย่างชัดเจน!

## 💻 การติดตั้งและรันโปรเจกต์ (Getting Started)

สำหรับเพื่อนร่วมก๊วน Dev โคลนโค้ดไปรันในเครื่องง่ายๆ ตามขั้นตอนนี้:

1. โคลน Repo ผ่าน GitHub
```bash
git clone https://github.com/your-username/portfolio.git
```
2. โหลดแพ็กเกจ (Dependencies)
```bash
npm install
```
3. รันเซิร์ฟเวอร์จำลองเพื่อพัฒนา (Dev Server)
```bash
npm run dev
```
4. กดเข้าไปที่ `http://localhost:5173` เพื่อดูผลลัพธ์เวทมนตร์แห่งโค้ดคุณได้เลย!
